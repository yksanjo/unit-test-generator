# Unit Test Generator

A focused unit test generator that creates comprehensive unit tests for Python functions and classes.

## Features

- AST-based code analysis
- Automatic edge case detection
- Function and class test generation
- pytest-compatible test files

## Usage

```bash
python unit_test_generator.py --source /path/to/code --output ./tests
```

## Installation

```bash
pip install -r requirements.txt
```
